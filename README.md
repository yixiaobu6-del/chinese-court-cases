# 中国裁判文书网浏览器

[![HTML](https://img.shields.io/badge/HTML-standalone-orange)]()
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-CSS-blue)]()

中国法院裁判文书浏览与分析工具。

## 功能

- **案例库**：100 个真实裁判案例
- **收藏系统**：收藏感兴趣的案例
- **案例对比**：3 个案例 × 10 个维度对比
- **搜索筛选**：按案由/法院/日期筛选

## 使用

打开 `web/browser.html` 即可使用。
